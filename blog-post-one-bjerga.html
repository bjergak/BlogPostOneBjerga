How to use Pandoc

Creating HTML Files and Converting a File from Markkdown to HTML

This article will teach you how to do two things: how to use the text editor on Pandoc to create a markdown file, and how to convert a pre-made markdown file into an html file.

Prerequisites

To use Pandoc, you will have to install it on your computer. It can be installed using your terminal. If you have not installed Pandoc yet, you can follow the instructions here.

Creating a HTML file using Pandoc

Step 1: Open a terminal and go to Directory

Open the terminal of your choosing. Once in the terminal, check to see where you are (directory-wise). This can be done by typing:

pwd

If your are using Windows, type:

echo %cd%

And hit enter. For this tutorial, we will be navigating to the Documents directory. To do this, type:

cd Documents

And hit enter. To make sure you are in the correct directory, type pwd (or echo %cd%) to check.

Now that you are in the Documents directory, create a subdirectory. In this case, we can call it “pandoc-test.” To create this subdirectory, type the following into your command prompt:

mkdir pandoc-test

And hit enter. Change to this directory by typing:

cd pandoc-test

You can check to make sure you are in the correct directory by typing pwd or echo %cd%. Now you are ready to create a Markdown file!

Step 2: Using Pandoc

In your terminal, type the following to open Pandoc:

pandoc

Hit enter. You should see a cursor on your screen, waiting for you to type something. Type something following the Markdown format. For example:

Hello *world*!

-What's
-Up?

When you have finished typing what you want, type Ctrl-D, or Ctrl-Z followed by Enter on Windows. After this is done, your text will be converted to HTML!

By default, Pandoc filters Markdown files into HTML files. However, you can change the order that it is filtered. For example, you can convert HTML files to Markdown files by entering the following:

pandoc -f html -t markdown

Then you can type HTML code, hit Ctrl-D (or Ctrl-Z then Enter on Windows), and it will be converted to Markdown.

Now let’s learn how to convert a pre-made Markdown file to HTML!

Converting Markdown File to HTML File

Converting a Markdown file to a HTML file is really easy! Just follow these two easy steps!

Step 1: Go to your Terminal

Make sure you are in the correct directory. Once you are in the correct directory, you will need to make sure your Markdown file is there. This can be done by typing:

ls

If you are using a Windows device, type the following:

dir

If you see the file that you created, you are on the right track!

Step 2: Convert the File

Now that you are in the location of your Markdown file, you are ready to convert it to a HTML file! To convert the file, type this command:

pandoc yourfile.md -f markdown -t html -s -o yourfile.html

The -s option tells Pandoc to create a “standalone” file. The -o yourfile.html tells Pandoc to put the output in the file “yourfile.html.” -f markdown and -t html could have been left out, since the default is to convert from Markdown to HTML, but it is always good practice to include them.

In this case, “yourfile” would be the name of the file you made.

Step 3: Check for the File

To make sure that Pandoc successfully converted your file, check to see that the HTML file was created. Look in your directory to find the file. This can be done by typing:

ls

or:

dir

If you are on Windows. Once you have confirmed the location of your file, you can open your HTML file in a browser. To do this, type the following into your command prompt:

open yourfile.html

If you are on Windows, type:

.\yourfile.html

If your file opens in a browser, SUCCESS! You have officially converted a Markdown file to a HTML file!

Conclusion

Pandoc is a very useful tool! It helps save time and avoids the tedious HTML coding!
